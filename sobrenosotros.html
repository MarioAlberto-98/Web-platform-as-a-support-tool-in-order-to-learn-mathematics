Titulo del Proyecto

Desarrollo de una plataforma web como herramienta de apoyo al aprendizaje de matemáticas (sumas, restas, multiplicaciones) en alumnos de tercer grado de la primaria Ignacio Rodríguez Galván durante el periodo 2020-2021














Plantamiento del problema

Desde hace años la dificultad de aprender conceptos con respecto a asignaturas de razonamiento matemático y aritmética en niños de primaria se ha visto notable, tal vez por el hecho de que se considera una materia compleja o difícil desde temprana edad, provocando el atraso de los estudiantes conforme van pasando los años, tendiendo dudas e incluso la existencia de problemas en la materia, provocando un rechazo social.










Objetivo General

Desarrollar una plataforma web como herramienta de apoyo al aprendizaje de las matemáticas (sumas, restas, multiplicaciones), aplicando los estilos de aprendizaje mediante juegos y actividades lúdicas, haciendo uso de las plataformas de desarrollo Unity, HTML, CSS, PHP, XAMPP y Java Script para alumnos de segundo grado de la primaria Ignacio Rodríguez Galván durante el periodo 2020-2021








Objetivos Especificos

• Diseñar una interfaz con un uso y diseño adecuando mediante las plataformas de desarrollo HTML y CSS para niños de segundo grado, escolaridad primaria.

• Estructurar un sistema de control de una base de datos en XAMPP para almacenar los datos del usuario, información y avances del estudiante.

• Estructurar un examen tipo prueba mediante PHP y Java Sript para medir el nivel de progreso del estudiante.

• Analizar de forma detallada las diferentes estrategias de enseñanza y aprendizaje en el área de Matemáticas mediante ejercicios interactivos lúdicos que ayuden a su comprensión.
